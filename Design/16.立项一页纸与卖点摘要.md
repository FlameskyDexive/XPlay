# 16. 立项一页纸与卖点摘要

## 1. 一句话定位

《荒天记》是一款 **轻度修仙放置 + 轻度异步SLG** 游戏，核心体验占比为 **修仙80% / SLG20%**：

- 玩家每天用 `10-30分钟` 完成收菜、构筑、渡劫、破境。
- 玩家不参与SLG，也能持续推进境界、刷个人挑战、建立长期目标。
- 宗门与SLG只提供效率放大、协作荣誉和服内故事，不绑主线命门。

## 2. 核心卖点

### 2.1 卖点一：修仙主循环完整闭环

- 不是单纯挂机收菜，而是“放置修炼 -> 构筑调整 -> 渡劫突破 -> 新内容解锁 -> 更高效率修炼”的完整循环。
- 每一层小境界都能配置雷劫/天劫，让成长节点具备操作峰值和情绪记忆点。
- 即使完全不进宗门，也能通过主线、试炼、遗迹、天梯、洞府经营长期游玩。

### 2.2 卖点二：轻社交，不绑架

- 宗门玩法以异步协作、战报托管、预约参与为主，避免强在线和重外交。
- 宗门收益偏效率、荣誉、外观，不提供主线唯一资源。
- 想社交的玩家有组织目标，不想社交的玩家也不会掉出主线竞争区间。

### 2.3 卖点三：轻量放置，也有构筑深度

- 功法、法宝、铭纹、道心、灵兽共同形成渡劫构筑。
- 玩家不是单纯堆数值，而是通过抗性、回复、爆发、控制等不同思路过劫。
- 中后期通过遗迹词条池、心魔机制和个人榜维持长线乐趣。

## 3. 目标用户

- 喜欢东方玄幻、修仙题材、境界成长和破境爽感的玩家。
- 喜欢放置和挂机，但不接受无脑数值堆叠的玩家。
- 接受轻社交、低压力宗门协作，但不愿被固定窗口和重指挥绑住的玩家。
- 玩过放置、卡牌、SLG，但希望“更轻、更稳、更能长期玩”的用户。

## 4. 产品红线

- 不参加SLG，不掉主线强度，不断主进度。
- 主线章节、境界突破、关键养成材料必须都有纯PVE来源。
- SLG强制操作时长不得超过总日常时长的 `20%`。
- 新手前7天不能出现“不加宗门就明显掉队”的体验。
- 任何一个版本，都必须回答“如果玩家完全不玩SLG，这个版本是否仍然成立”。

## 5. 日常体验模型

### 5.1 轻度日常

- `10-15分钟`：上线收菜、处理一次突破准备、打一两个短挑战、领每日奖励。

### 5.2 标准日常

- `20-30分钟`：完成收菜、构筑微调、挑战雷劫/试炼/遗迹，并处理少量宗门托管操作。

### 5.3 深度游玩

- `45分钟+`：冲高难雷劫、刷遗迹极限构筑、参与宗门周结算或协作Boss。

## 6. 不玩SLG为什么还能长线玩

### 6.1 7日内

- 完成修炼、洞府、突破、雷劫的完整闭环。
- 建立第一套可感知的功法/法宝/道心构筑。
- 解锁至少 `2-3` 个中期追求入口。

### 6.2 30日内

- 形成“破境 -> 刷本 -> 重构 -> 再破境”的稳定中期循环。
- 拥有个人周常、个人榜、遗迹词条池、洞府巡修等稳定目标。
- 即便不加宗门，依然能看到明确战力成长和系统解锁。

### 6.3 90日以上

- 通过赛季词条轮换、个人极限挑战、图鉴和外观收藏维持长期追求。
- 通过不同流派构筑维持内容复玩，而不是只靠线性堆数值。

## 7. SLG为什么要保留20%

- 提供服内组织感、协作感和谈资，放大游戏世界的“活人感”。
- 给深度玩家和公会型玩家提供额外舞台，但不抢主循环。
- 用周结算、战报、荣誉和外观承接“炫耀与归属”，而不是承接“主线必需成长”。

## 8. 首发版本必须成立的内容

- 境界/修炼/突破/雷劫闭环。
- 洞府与离线收益体系。
- 主线章节 + 境界试炼 + 雷池秘境 + 遗迹/天梯中的至少两类中期内容。
- 失败补偿、反卡关与纯修仙周目标体系。
- 宗门软接入与奖励边界守护。

## 9. 商业化原则

- 付费卖效率、便捷和外观，不卖主线唯一资格。
- 纯修仙用户也要有稳定付费点：挂机便捷、洞府装饰、法宝特效、成长容错。
- 宗门相关付费不得制造“不参加SLG就吃亏”的明显感知。

## 10. 版本成功判断

- 纯修仙玩家30日留存达到版本目标。
- 不参与SLG玩家的境界进度保持在参与SLG玩家的 `85%-90%`。
- `slg_time_share_p90` 长期不高于 `25%`。
- 玩家反馈中，核心认知应是“这是一个能长期玩的修仙游戏”，而不是“一个披着修仙皮的SLG”。

## 11. 给团队的执行提醒

- 研发资源先保修仙主链，再接SLG副链。
- 运营活动先写纯修仙版本，再考虑附加宗门层。
- 策划新增系统时，先判断是否能服务纯修仙长线目标。
- 程序和数值上线前，必须检查奖励边界、来源占比和时长占比。
