## 一、文档目标

本版本用于明确《荒天记》的产品主轴：**修仙主玩法长期可玩，SLG副玩法可选增益**。

本次重构重点：
- 强化“**不玩SLG也能长线玩**”的完整闭环。
- 修仙部分改为“**放置+构筑+挑战**”三线并行。
- 境界体系参考《完美世界》分层思路，形成完整成长曲线。
- **每一层小境界突破都配置雷劫/天劫机制**，避免纯数值点点点。
- 将“修仙80%、SLG20%”明确落实到内容、时长、奖励边界与版本节奏中。

---

## 二、产品定位

### 2.1 核心定位
- 游戏类型：轻度修仙 + 轻度SLG。
- 目标受众：喜欢东方玄幻成长、可挂机、低社交压力玩家。
- 核心结构占比：修仙体验 `80%`，SLG体验 `20%`；该比例同时约束内容供给、在线时长与奖励投放。
- 核心承诺：
  - 不参加SLG，不掉主线强度，不断主进度。
  - 每天10-30分钟可完成核心成长动作。
  - 离线收益与在线操作都有效，兼容碎片化体验。

### 2.2 玩法优先级
1. 修仙（主线，PVE可独立完整体验）
2. 宗门/SLG（副线，提供资源效率和社交荣誉）
3. 收集养成（英雄、灵兽、法宝、功法图鉴）

### 2.3 设计红线（必须满足）
- 主线境界突破、主线章节、关键养成材料都提供纯PVE来源。
- SLG产出不绑定“唯一核心材料”，仅提供效率、外观、荣誉增益。
- 玩家可永久不加入宗门战，不会卡在核心成长门槛。
- 任何时期都要有可挂机目标、可短时操作目标、可挑战目标。
- SLG强制操作时长不得超过总日常时长的20%，且不能构成固定窗口在线压力。

## 三、世界观与叙事方向

### 3.1 世界观基调
- 参考《完美世界》的荒古修行气质：下界、上界、仙域、帝路。
- 核心叙事主题：凡骨逆天、渡劫登临、以身证道。
- 叙事结构：主线章节 + 奇遇支线 + 境界传记。

### 3.2 剧情分层
- 下界篇：宗族、荒域、初登修行路。
- 上界篇：天骄争锋、古教冲突、法则初窥。
- 帝关篇：异域征伐、绝境破境、道心试炼。
- 仙域篇：大道重塑、仙王争锋、终局证帝。

说明：剧情风格与境界节奏参考经典玄幻体验，具体人物与剧情事件采用原创化改编，保证长期内容扩展空间。

## 四、总玩法结构

### 4.1 双主循环

**A. 纯修仙循环（强制可闭环）**
1) 挂机修炼/洞府产出 -> 2) 功法构筑 -> 3) 小层突破渡劫 -> 4) 新副本/新系统解锁 -> 5) 获取更高阶资源 -> 回到1。

**B. SLG循环（可选增益）**
1) 宗门经营 -> 2) 领地扩张 -> 3) 宗门协作活动 -> 4) 获取效率加速与荣誉奖励 -> 反哺A循环。

### 4.2 玩家日常时间模型
- 轻度日常（10-15分钟）：收菜、点修炼、处理1次渡劫、清日常任务。
- 标准日常（20-30分钟）：额外打2-3个副本、调整功法、挑战天梯。
- 深度游玩（45分钟+）：冲击高难雷劫、刷遗迹、参与宗门协作（可选）。

### 4.3 不参与SLG玩家的长线目标链
- 7日目标：完成修炼、洞府、突破、雷劫闭环，并解锁至少2个中期追求入口。
- 30日目标：完成至少2次大境跨越，形成“破境 -> 刷本 -> 重构”的稳定循环。
- 90日以上目标：通过赛季词条池、遗迹轮换、个人榜与图鉴收藏维持长线追求。
- 详细约束见：[`15.纯修仙长线目标与SLG解耦原则`](./15.纯修仙长线目标与SLG解耦原则.md)。

## 五、修仙主系统（核心）

## 5.1 境界体系（参考《完美世界》层级）

采用“**大境界 + 9重小层**”统一结构，便于放置成长和长期内容配置。

|---|---|---:|---|

| 凡道 | 洞天境 | 1-9重 | 洞府系统、首个法宝位 |
| 凡道 | 化灵境 | 1-9重 | 灵兽系统、灵技共鸣 |
| 凡道 | 铭纹境 | 1-9重 | 铭纹盘、词条养成 |

| 神道 | 神火境 | 1-9重 | 神火淬体、高阶材料副本 |

> 总计 18 个大境界，162 次小层突破，天然形成长期目标池。

## 5.2 每层突破雷劫/天劫机制（刚性要求）

### 5.2.1 触发规则
- 每次从 `N重 -> N+1重` 的突破都必触发一次雷劫。
- 每个大境界的 `9重 -> 下一大境界1重` 必触发“天劫 + 心魔劫”双劫。
- 玩家可手动挑战，也可选择自动渡劫（成功率较低，适配放置用户）。

### 5.2.2 劫难结构
- 雷劫伤害轴：雷击段数、雷伤倍率、穿透。
- 道心干扰轴：沉默、逆脉、心魔幻象。
- 环境惩罚轴：灵气紊乱、治疗衰减、护盾衰减。

### 5.2.3 小层雷劫模板（每境通用）
| 小层突破 | 劫名 | 雷击段数 | 附加机制 |
|---|---|---:|---|
| 1->2 | 一九引雷劫 | 9 | 无 |
| 2->3 | 二九裂脉劫 | 18 | 轻度破防 |
| 3->4 | 三九灼魂劫 | 27 | 持续灼伤 |
| 4->5 | 四九蚀灵劫 | 36 | 灵力恢复降低 |
| 5->6 | 五九噬元劫 | 45 | 护盾效率降低 |
| 6->7 | 六九破道劫 | 54 | 技能冷却延长 |
| 7->8 | 七九镇神劫 | 63 | 控制抗性考验 |
| 8->9 | 八九覆天劫 | 72 | 高穿透爆发 |
| 9->下境1 | 九九天劫+心魔劫 | 81 | 双阶段首领战 |

### 5.2.4 失败与补偿
- 失败不降大境界，只回退当前小层进度。
- 返还80%突破资源，避免挫败过强。
- 失败获得“劫灰”，可兑换：
  - 下一次雷劫减伤
  - 一次性保命符
  - 心魔抗性药
- 连续失败触发“天道垂怜”软保底：最多降低50%雷劫强度。

### 5.2.5 渡劫策略位（构筑深度）
- 法宝位：抗雷、反震、护命。
- 丹药位：瞬回、持续回复、免控。
- 阵法位：减伤阵、聚灵阵、净心阵。
- 护道人位：英雄/灵兽提供一次外援技能。

## 5.3 放置修炼体系

### 5.3.1 核心放置模块
- 打坐：基础修为稳定产出，离线累计上限24小时。
- 闭关：消耗资源换高倍修为，适合冲境界节点。
- 游历：派遣式玩法，产出稀有材料与奇遇事件。
- 洞府：建筑化成长，提供常驻增益与自动产出。

### 5.3.2 产出公式（示意）
```
每小时修为 = 境界基础值 × 功法系数 × 洞府系数 × 奇遇加成 × (1 + 离线补偿)
```

### 5.3.3 放置友好设计
- 离线收益不低于同阶段在线收益的70%。
- 每天首次登录给予“悟道时刻”短时加速，缓解补课压力。
- 自动修炼队列支持12小时计划，降低重复点击。

## 5.4 PVE内容池（纯修仙可长期游玩）

- 主线章节：推进世界观和系统解锁。
- 境界试炼：每个大境界独立Boss链，绑定突破前置挑战。
- 雷池秘境：专产渡劫材料、抗雷法宝胚子。
- 太古遗迹：Roguelike地图，提供随机词条构筑乐趣。
- 无尽天梯：长期排行，按个人战力分段，不绑定SLG。
- 心魔幻境：高风险高收益模式，产出稀有铭纹。

## 5.5 养成系统（修仙主线侧）

- 功法：炼体/炼气/炼神三体系，可双修但有冲突成本。
- 法宝：主动技 + 被动词条，支持洗炼与继承。
- 灵兽：战斗辅助与资源增益双定位。
- 铭纹：中后期核心数值系统，绑定流派差异化。
- 道心：影响雷劫抗性与心魔成功率，避免纯堆攻击。

## 5.6 长线内容节奏

- 日循环：修炼、渡劫、短副本、资源回收。
- 周循环：境界冲刺、天梯结算、遗迹轮换。
- 赛季循环（4-6周）：新增词条池、主题Boss、赛季目标。
- 永久循环：境界总进度、图鉴、成就、外观收藏。

## 六、SLG副系统（可选，不绑主进度）

## 6.1 定位原则
- SLG是“效率放大器”和“社交荣誉场”，不是主线门槛。
- 不做重外交、重在线强绑定玩法。
- 不参加SLG仍可拿到全部主线成长关键资源。

## 6.2 玩法内容
- 宗门建设：主殿、藏经阁、灵田、矿场、防御阵等。
- 轻领地战：低频开战、自动战报、可预约参与。
- 宗门协作Boss：异步集火，降低同时在线要求。
- 宗门任务：捐献、护送、采集等轻任务。

## 6.3 奖励边界（关键约束）
- SLG奖励以以下类型为主：
  - 修炼效率加速道具
  - 外观、称号、社交展示
  - 非唯一可替代的资源包
- 禁止将“境界突破必要材料”设为SLG独占来源。

## 6.4 负担控制
- 战报与调度支持全自动托管。
- 建筑升级时间短，减少等待惩罚。
- 保护机制：新手免战期 + 低损撤退规则。

## 6.5 占比与门槛控制
- 以20-30分钟标准日常计算，SLG强制操作时长控制在0-6分钟。
- 首周SLG只做软开放，不作为主线漏斗关键节点。
- 不参加SLG的玩家以个人周常、遗迹挑战、洞府巡修承接中期目标，不补发荣誉类奖励。

## 七、经济与数值框架

## 7.1 资源分层
- 核心成长资源：修为、灵气、渡劫材料、功法残卷（PVE稳定产出）。
- 效率资源：加速符、生产增益、宗门贡献（SLG可加成）。
- 荣誉资源：称号点、赛季勋章（展示为主）。

## 7.2 战力来源占比目标
- 修仙主系统：70%-80%
- PVE装备/法宝/铭纹：15%-25%
- SLG相关增益：<=10%

## 7.3 境界需求曲线（示意）
```
小层需求修为 = A(境界) × 层数系数 × 成长倍率
大境界跃迁需求 = 当前大境9重需求 × 2.5~4.0
```

## 7.4 雷劫强度曲线（示意）
```
雷劫强度 = 基础强度(境界) × 小层系数 × (1 - 失败补偿系数)
```

## 7.5 反卡关机制
- 主线卡关超过48小时触发“天机指引”任务链，给定向突破材料。
- 境界突破连续失败触发保底增益，不出现长期硬卡。
- 所有关键养成都有替代来源，避免单点堵塞。

## 八、系统实现与ET落地

说明：若希望降低包数量，可将 `tribulation` 并入 `cultivation`，但配置与协议应逻辑分层。

## 8.2 依赖原则
- 严格遵守项目现有包层级，不做反向依赖。
- 修仙主链依赖 `unit/numeric/spell/item/quest/equipment`。
- SLG链依赖 `map/move/aoi`，不得反向卡死修仙主链。

## 8.3 关键配置表（Luban/Excel）
- `RealmConfig`：大境界与小层参数。
- `RealmBreakConfig`：每层突破修为需求。
- `TribulationConfig`：每层雷劫波次、机制、奖励。
- `GongfaConfig`：功法属性、修炼效率、流派标签。
- `CaveConfig`：洞府建筑与挂机增益。
- `AdventureStageConfig`：主线章节与掉落。
- `SLGRewardLimitConfig`：SLG奖励边界配置。

## 8.4 协议建议（示意）
- 修炼：`C2M_CultivateStart` / `M2C_CultivateSync`
- 突破：`C2M_RealmBreakTry` / `M2C_RealmBreakResult`
- 雷劫：`C2M_TribulationEnter` / `M2C_TribulationSettle`
- 洞府：`C2M_CaveCollect` / `M2C_CaveStatus`
- 宗门：`C2M_SectAction` / `M2C_SectBattleReport`

## 九、开发优先级与里程碑

## 9.1 阶段一（最高优先级）：纯修仙闭环
目标：不含SLG也可玩30天以上。

任务：
1. 境界+修炼+突破+雷劫闭环。
2. 洞府放置与离线收益。
3. 主线章节1-2章与基础副本。
4. 反卡关与失败补偿机制。

验收：新账号7日内完成“多次小层突破 + 至少一次大境跨越”。

## 9.2 阶段二：修仙深度与长线内容
目标：提升构筑深度与中后期黏性。

任务：
1. 功法流派与铭纹系统。
2. 雷池秘境、无尽天梯、心魔幻境。
3. 赛季词条池与阶段目标。

## 9.3 阶段三：轻SLG接入
目标：作为副玩法接入，不影响主线独立性。

任务：
1. 宗门创建、建筑、轻领地玩法。
2. 协作Boss与战报托管。
3. 奖励边界校验（不产唯一核心材料）。

## 9.4 阶段四：整合与运营化
目标：稳定、平衡、可持续更新。

任务：
1. 平衡性与经济回收。
2. 活动模板化与内容流水线。
3. 性能优化与监控告警。

## 十、商业化与公平性

### 10.1 付费点建议
- 月卡/战令：离线上限、便捷功能、外观权益。
- 外观类：角色皮肤、法宝特效、洞府装饰。
- 资源类：非唯一资源包、限次购买。

### 10.2 公平性原则
- 不售卖“突破必需唯一材料”。
- 不以SLG活跃度决定主线强度上限。
- 付费提升效率，不直接越级碾压。

## 十一、关键风险与应对

### 11.1 纯放置疲劳风险
- 风险：长期挂机导致体验单调。
- 对策：雷劫策略位、Roguelike遗迹、赛季词条刷新。

### 11.2 境界卡关挫败风险
- 风险：连续失败导致流失。
- 对策：失败返还、劫灰兑换、软保底、指引任务。

### 11.3 SLG反客为主风险
- 风险：副玩法绑架主进度。
- 对策：奖励边界配置 + 数据巡检 + 版本前校验。

### 11.4 数值通胀风险
- 风险：资源无限膨胀，目标感下降。
- 对策：多层资源池、回收机制、赛季重置部分临时资源。

## 十二、验收指标（围绕“不玩SLG也能长线玩”）

### 12.1 核心指标
- `纯修仙玩家30日留存`：达到立项预期阈值。
- `纯修仙玩家日活时长`：稳定在目标区间。
- `未参与SLG玩家的境界进度`：不低于参与SLG玩家的80%-90%。

### 12.2 体验指标
- 7日内至少经历10次以上小层雷劫。
- 14日内至少完成2次大境界跨越。
- 30日内仍存在清晰中期目标（境界、法宝、天梯、图鉴）。

### 12.3 负反馈监控
- 连续卡关 > 72小时用户占比。
- 雷劫失败后次日流失率。
- 因SLG压力导致退坑的反馈占比。
